Cargo Noir 黑货船

Posted on May 24, 2017 本文总阅读量次

黑货船 Cargo Noir

介绍

  • 游戏人数 : 2 - 5

  • 推荐人数 : 5

  • 游戏时长 : 30 - 90 mins

  • 游戏重度 : 2.08 (轻策、毛线)

  • 游戏类型 : 家庭聚会

  • 游戏机制 : 行动、拍卖 | 集合收集 | 工人放置

  • 当前排名 : 1399

  • 综合评分 : 6.5

    在Serge Laget的黑货船（他的第四款Day of wonders的独立大盒游戏）中，玩家扮演50年代在水路上交易拥有黑色背景的“家族”。在每回合中，您将会到达各种港口，香港，孟买，鹿特丹，纽约等等。在这里，货物被称为以适当的价格“丢失”，而您将会为“丢失展出”的货物提供报价。如果另一个家族在那个港口提供了更高的报价，那么你需要提高出价或者滚蛋，溜走去寻找其他地方的货物。虽然你只是站在港口上，但是您却能够将货物顺利地从码头移动到您的个人仓库。一旦你收集了货物，你就可以交易他们，以增加更多的船只到您的船队，从而允许您在更多的地方侦察货物；购买胜利点数；或采取其他行动。你收集的商品越多，它们就越有价值。游戏结束时拥有最多价值的玩家获胜。

游戏流程

Setup

  1. 每个玩家获取一张家族成员卡（除了颜色和图案没有其他区别）

  2. 每个玩家获得7金币和3个对应家族的小船（颜色）

  3. 根据玩家人数设置港口板块

  4. 将玩家数量张百搭（wild）与常规货物一起放进抽取袋，摇匀后抽取货物填满港口货物空槽。

  5. 选择一个先手玩家开始游戏

Phases

从先手玩家开始，依次执行这三个行动。特别的，第一回合，玩家跳过前两个阶段。最后一个回合，玩家跳过第三阶段。

  1. 收回所有未遭遇抵抗的小船并且获取上面的货物。如果你的小船遭遇更高的报价（outbid），你可以选择增加你的筹码直到超过对手，或者是回收小船以及筹码并且重新排遣到一个新的地区（本轮不能再放置在同一个地点）。

    被获取的货物立刻补充（从袋中重新抽取）

  2. 购买卡牌。你可以通过支付任意组合的获取进行购买功能卡牌或者分数卡（组合价值在玩家板块下方，分使用不同种类以及使用相同种类两张方式），蓝色的是你游戏结束获得的分数，红色的是价格。你可以选择保留货物，在你的回合结束你必须丢弃超过你仓库容量部分的货物。

    • 仓库：提供额外货物的存储空间 （限制2个）
    • 货船：提供额外的小船（即行动点，限制2个）
    • 打手：当你被outbid的时候，你将获取2金币。在你的回合每个港口每个打手只能使用一次。
    • 除了仓库（warehouse）货船（cargo ships）以及打手（syndicate），其他卡牌都只提供游戏结束分数。

      注意，如果这是最后一个回合，用作于购买时，你每个金币相当于1点价值。
      使用完的货物不扔回抽取袋。胜利点数是公开信息。

  3. 派遣出你的小船。有三种地区可以去：

    • 黑市（Macao）：你可以选择将仓库的一个货物与一个展示的货物交互；或者你可以在袋中抽取一个货物加入仓库。
    • 赌场（Casino）：这里是你唯一拿钱的地方，每个小船获取2元。
    • 常规港口：放下小船的同时押下你觉得地区提供货物的报价（金币）。

      一次执行完所有小船

  4. 维持。从货物抽取袋中抽取并重新填充这轮你拿取货物的地区。如果你是先手玩家的话，将回合标记推后一格。

End of Game

游戏到达指定轮数时，执行最后一次行动即游戏结束。

  • 2-3人游戏：11轮
  • 4-6人游戏：10轮

Winner

所有商品价值加起来最高的获得游戏胜利！
平局的话看说明书。

策略提醒 : 被outbid是较大损失，直接损失一个回合，宁愿一开始就下较多而准确的筹码，而不是慢慢加上去。
